CustomKnits Docs

Source code for the CustomKnits documentation website (https://docs.customknits.ru
).
Built with Docusaurus.

Install
npm install

Develop
npm start


Runs a local dev server with live reload.

Build
npm run build

Deploy

Deployment is automated via GitHub Actions (published to gh-pages).
Manual deploy:

GIT_USER=<your-username> npm run deploy

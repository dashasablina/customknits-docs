# Введение

CustomKnits — это визуальный редактор вязания и генератор узоров, созданный для удобного проектирования схем и расчётов.

Здесь собрана документация о том, как пользоваться проектом:  
как создавать узоры, настраивать пряжу и плотность, управлять проектами и экспортировать результат.

## Что вы найдёте в документации

- основы работы с редактором схем;  
- использование генератора узоров;  
- работа с пряжей, плотностью и расчётами;  
- создание собственных пресетов и шаблонов;  
- экспорт проектов (PNG, SVG);  
- дополнительные настройки.

## Для кого этот инструмент

- для начинающих и опытных вязальщиц;  
- для дизайнеров, которые хотят быстро создавать схемы;  
- для разработчиков, интегрирующих CustomKnits в свои проекты.

Интерфейс

Интерфейс CustomKnits состоит из трёх основных областей:

  • верхнее меню,
  • боковая панель,
  • рабочая область.

Каждый из этих элементов играет свою роль в редактировании выкроек и вязания.

Верхнее меню​

В строке верхнего меню вы видите:

  • логотип проекта — по нажатию вы перейдёте на главную страницу
    CustomKnits
  • название проекта — по нажатию можно задать новое имя
  • действия уровня проекта (см. ниже)
  • настройки проекта (см. ниже)

Действия уровня проекта​

В этом меню находятся команды, относящиеся ко всему проекту либо действия для экспорта выкроек или деталей вязания:

  • создание нового проекта
  • открытие существующего
  • сохранение проекта
  • экспорт выбранной выкройки
  • экспорт выбранного вязания

Настройки проекта​

В меню настроек вы можете:

  • выбрать язык интерфейса (доступны русский и английский)
  • сбросить панели интерфейса рабочей области в стартовое положение
  • настроить внешний вид рабочей области
  • специфичные для рабочей области настройки

Обратите внимание:

Меню настроек относится к той рабочей области, которая открыта сейчас.
Если открыта выкройка — будут настройки для выкройки.
Если открыто вязание — будут настройки для вязания.

Подробности для каждой рабочей области будут описаны в соответствующих разделах ниже.

Боковая панель​

Боковая панель — это центральное место для управления проектом.
Здесь отображаются все выкройки, все вязаные детали, заметки проекта, а также действия, которые можно выполнять с ними.

Управление панелью​

  • Вы можете менять ширину панели — потяните за вертикальный ползунок справа.
  • Панель можно свернуть полностью — нажмите на стрелочку в левом верхнем углу.

Список элементов проекта​

В панели отображается полный перечень объектов текущего проекта:

  • выкройки,
  • вязаные детали,
  • заметки.

Чтобы добавить новую выкройку или вязание, наведите курсор на заголовок списка — появится кнопка "+".

Действия с элементами списка​

Когда вы наводите курсор на конкретный элемент в списке, появляются дополнительные действия:

  • Превью
  • Скопировать
  • Удалить

Для элементов с визуальным содержимым доступно превью, позволяющее быстро оценить деталь, не открывая её.

Рабочая область​

Рабочая область — это основное пространство, где вы редактируете выкройки и вязаные детали.
У рабочей области выкроек и вязания есть общие элементы управления, но есть инструменты, которые отличаются.

Подробное описание специфических инструментов для каждого типа будет приведено в отдельных разделах.

Управление рабочей областью​

Перемещение​

  • Зажмите пробел и перемещайте рабочую область, удерживая левую кнопку мыши.
  • На macOS можно перемещать двумя пальцами по трекпаду — как обычное панорамирование.

Масштабирование​

  • Используйте клавиши + и – или колесо мыши.
  • При удерживании Ctrl колесо меняет масштаб,
    без Ctrl — перемещает доску по горизонтали и вертикали.

Вписать в экран​

Команда F автоматически располагает всю выкройку или деталь вязания так,
чтобы она полностью помещалась в окно редактора.

Сброс масштаба и положения​

Кнопка R возвращает масштаб к значению 100% и сбрасывает положение рабочей области на стандартное.

История действий​

Отмена и возврат действий доступны в виджете с двумя стрелками — назад и вперёд.

Перемещение панелей​

Панели интерфейса можно перетаскивать, меняя их расположение.
Если расположение панелей стало неудобным, вы можете выполнить команду:

Настройки → Сбросить панели интерфейса,
чтобы вернуть их в исходное состояние.

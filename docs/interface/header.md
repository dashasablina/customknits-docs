# Верхнее меню

Верхнее меню предоставляет быстрый доступ к действиям уровня проекта и настройкам редактора.  
Через него можно создавать и открывать проекты, управлять сохранениями и экспортом, а также изменять параметры рабочей области.

В верхней части интерфейса расположены:

- **меню с действиями уровня проекта** (описаны ниже)
- **логотип проекта** — по нажатию открывается главная страница  
  [CustomKnits](https://customknits.ru)
- **название проекта** — можно переименовать по клику
- **настройки проекта** (описаны ниже)

В этом меню находятся команды, которые относятся ко всему проекту:

- создание нового проекта  
- открытие существующего  
- сохранение проекта  
- экспорт выбранной выкройки  
- экспорт выбранного вязания  

---

### Настройки проекта

<img
  src={require('/img/interface/generalSettings.png').default}
  srcSet={require('/img/interface/generalSettings.png').default + " 2x"}
  alt="Настройки проекта"
/>

В меню настроек доступны параметры, влияющие на интерфейс и рабочую область:

- выбор **языка интерфейса** (русский или английский)
- сброс панелей рабочего пространства в исходное положение
- настройка внешнего вида рабочей области
- параметры, специфичные для текущего режима (выкройка или вязание)

Обратите внимание:

**Показанные настройки относятся к той рабочей области, которая открыта в данный момент.**  
Если открыта выкройка — отображаются настройки выкройки.  
Если открыто вязание — показываются настройки вязания.

Подробная информация по инструментам каждой рабочей области приведена в отдельных разделах документации.

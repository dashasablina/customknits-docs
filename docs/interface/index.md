# Интерфейс

В этом разделе описан общий интерфейс редактора **CustomKnits**: его основные элементы, базовые принципы работы и управление.  
Сначала мы рассмотрим общие компоненты интерфейса — верхнее меню, боковую панель и рабочую область.  

После этого в отдельных разделах будут подробно описаны инструменты и возможности двух видов рабочих областей:

- **редактор выкроек**,  
- **редактор вязания**.

Этот обзор поможет быстро ориентироваться в приложении и понять, где находятся ключевые функции.
